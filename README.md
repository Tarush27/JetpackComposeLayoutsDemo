Pokémon Characters Collection

Android app demonstrating the usage of different layouts built using Composables, as stated below, using Jetpack Compose.

- Row
- Column
- Lists (Lazy Row)
- Modifiers
- Grids (Lazy Horizontal Grid)
- Navigation Bar and Navigation Rail
- TextField, Text, and Image, Icons
- Slot Apis (Scaffold)
- Material Theme
